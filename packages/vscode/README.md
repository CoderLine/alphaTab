# alphaTab Language Server

This is a Language Server (LSP) for writing alphaTex, providing coding assistance to code editors.

Beside the full language server, this package also contains wider modules for adding alphaTex support to editors like:

* A TextMate grammar for basic tokenizing and syntax highlighting
* Technical/Reference documentation for alphaTex (for the use in documentation hints or showing user-help in editors).


Features:

* Basic Syntax Highlighting
* Hover Documentation 
* Code Completion 
* Signature Help for Value Lists 
* Formatter
